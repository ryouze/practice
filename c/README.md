## brute_force_pin
enter a PIN number of any length (e.g., 0021), then brute force it.

example:
```
Enter a PIN number: 003
OK: Received PIN to guess: 003 (3 total digits, 1 valid digits, 10 possible combinations, max loop value for digits: 10)
Proceed? [y/n]: y
-> Wrong: 000
-> Wrong: 001
-> Wrong: 002
<- Correct: 003
The PIN was: 003
```
