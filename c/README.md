## binary
convert decimal number to binary number.

example:
```
348 / 2 = 174 R 0
174 / 2 = 87 R 0
87 / 2 = 43 R 1
43 / 2 = 21 R 1
21 / 2 = 10 R 1
10 / 2 = 5 R 0
5 / 2 = 2 R 1
2 / 2 = 1 R 0
1 / 2 = 0 R 1
101011100
```

## brute_force_pin
enter a 4-digit PIN number (e.g., 0021), then brute force it.

example:
```
Enter a 4-digit PIN to guess: 0002
You entered: 0002
-> Wrong: 0000
-> Wrong: 0001
<- Correct: 0002
```

## heron
calculate the square root of a number using the babylonian / heron's algorithm

example:
```
// iter_no=1; g=7.042
// iter_no=2; g=5.296
// iter_no=3; g=5.008
// iter_no=4; g=5
// The square root of 25 is 5.
```
